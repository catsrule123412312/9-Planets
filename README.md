# 🌌 9-Planets Survival Game

Welcome to the 9-Planets survival sandbox! Follow these quick steps to launch the game instantly.

## 🚀 How to Play 

1. Click the green Code button at the top right of this page.
2. Select the Codespaces tab.
3. Click the green Create codespace on main button.
4. Wait about 10 seconds for your space to open. 

Once the terminal box pops up at the bottom of your screen, type this command and press Enter to start:


python3 9_planets.py
